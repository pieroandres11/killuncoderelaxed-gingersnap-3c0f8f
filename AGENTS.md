# AGENTS.md

## Arquitectura

Proyecto estático de una sola página (`index.html`) sin frameworks ni dependencias.

## Estructura

```
index.html   — página principal con contenido de texto
```

## Convenciones

- Todo el estilo está inline en `<style>` dentro del `<head>`.
- Sin JavaScript; sin build step requerido.
- Para agregar más páginas, crear archivos `.html` adicionales en la raíz.
